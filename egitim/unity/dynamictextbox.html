UNITY

Dynamic TextBox
Unity oyun motorunda text gibi UI elementlerini kullanabilmek için using UnityEngine.UI; kütüptanesini eklememiz gerekiyor.


Projemize kullanacağımız textbox'ı script sayfamızda boyutuyla oynama yapabilmek için public RectTransform olarak tanımlayıp kod sayfamız hangi objede ise o objenin inspector kısmındaki gözüken kod sayfamıza hierarcyh kısmından textbox'ımızı sürüklüyoruz.
TextBox'ımızın text'ine müdahale edebilmek için ise public Text olarak tanımlayıp kod sayfasına textbox'ı sürükleme işlemini yapıyoruz.

TextBox'ın boyutuna müdahale etmek için RectTransform için tanımladığımız değişkeni kullanarak

"dinamikyazıkutusu2.sizeDelta = new Vector2(dinamikyazıkutusu2.rect.width, dinamikyazıkutusu2.rect.height + buyut);"

kodu yazıyoruz.

"dinamikyazıkutusu2.rect.width" bu kod bize textbox'ımızın hali hazırdaki uzunluğunun tekrardan kendisine verilmesini sağlıyor.

"dinamikyazıkutusu2.rect.height + buyut" bu kod ise yüksekliğimize boyut değişkenimizin değeri kadar artmasını sağlıyoruz.

Kodu bu şekilde kullanırsak textbox'ımız boyut değişkenimizin yarısı kadar aşağı doğru, yarısı kadar yukarıya doğru büyüyecek. Bizim seçeneğimize sabit kalarak yukarı yada aşağı doğru büyümesini istersek

'dinamikyazıkutusu2.transform.position = new Vector2(dinamikyazıkutusu2.transform.position.x, dinamikyazıkutusu2.transform.position.y - buyut / 2);'

kodu yazarak boyut değişkeninin yarısı kadar aşağıya iterek sanki sabit kalmış da aşağıya doğru büyüme sağlamış oluyoruz.
TextBox'ın text'ine müdahale etmek için ise

'dinamikyazıkutusu.text += "Funny Coders" + "\n";'

kodunu kullanıyoruz.

(text değişkenimizin ismi).text = string ifade;

using System.Collections;
using System.Collections.Generic;
using UnityEngine;
using UnityEngine.UI;
public class dynamiictextbox : MonoBehaviour
{
float buyut = 45.5f;
public RectTransform dinamikyazıkutusu2;
public Text dinamikyazıkutusu;
void Start()
{
StartCoroutine(yaz());
}
IEnumerator yaz()
{
dinamikyazıkutusu.text += "Funny Coders" + "\n";
dinamikyazıkutusu2.sizeDelta = new Vector2(dinamikyazıkutusu2.rect.width, dinamikyazıkutusu2.rect.height + buyut);
dinamikyazıkutusu2.transform.position = new Vector2(dinamikyazıkutusu2.transform.position.x, dinamikyazıkutusu2.transform.position.y - buyut / 2);
yield return new WaitForSeconds(5f);
StartCoroutine(yaz());
}
}
