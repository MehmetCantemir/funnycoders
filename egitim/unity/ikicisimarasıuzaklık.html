UNITY

İki Cismin Mesafe Farkı
İki cismin farkını basit bir şekilde almamızı sağlayan kod : Vector3.Distance (birinci cisim posizyonu , ikinci cisim posizyonu);
Projemizde aralarındaki mesafe farkını alacağımız iki cismi bir kod sayfasına 'GameObject' olarak tanımladıktan sonra 'Vector3.Distance(1.cisminizinadınıyazın.transform.position,2.cisminizinadınıyazın.transform.position)

(Eğer cisimleri kod sayfanızda sadece konum farkını almak için kullanacaksanız transform olarak da tanımlayabilirsiniz)

using System.Collections;
using System.Collections.Generic;
using UnityEngine;
using UnityEngine.UI;
public class mesafe : MonoBehaviour
{
public Text mesafetext;
public GameObject topumuz;
[SerializeField] float aradakiuzaklik;
void FixedUpdate()
{
aradakiuzaklik = Vector3.Distance(topumuz.transform.position, transform.position);
mesafetext.text = System.Convert.ToString(aradakiuzaklik);
}
}
Projenin .exe dosyası için tıklayın.
Projenin unity dosyaları için tıklayın.
