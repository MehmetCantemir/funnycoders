C Programlama Döngüler

Programlamada belli işleri birden fazla tekrarlayacaksak eğer döngüleri kullanırız.
Basit bir şekilde örneklemek gerekirse bir sayıyı 100 defa yazmak veya 1'den 200'e kadar sayıları ekrana yazdırmak. Döngüleri kullanmadan bu örnekleri yapmak istersek çok fazla kod yazmamız gerekir ve bu hoş ve mantıklı olmaz. Döngülerle bu tarz işleri kısaltıyoruz.



FOR Döngüsü





#include <stdio.h>
int main(){
int i = 10;
for (i = 0 ; i < 10 ; i++ ){
printf("i = %d \n ",i);
}
return 0;
}

While Döngüsü




#include <stdio.h>
int main(){
int sayac = 0;
while (sayac = 0 ; sayac < 10 ; sayac++ ){
printf("sayaç = %d \n ",sayac);
sayac++;
}
return 0;
}

Do-While Döngüsü



#include <stdio.h>
int main(){
int sayac = 0;
do
{
printf(" %d \n ",sayac);
sayac++;
}while (sayac < 10);
return 0;
}
